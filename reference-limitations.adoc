既知の制限事項は、このリリースの製品でサポートされていないプラットフォーム、デバイス、機能、または製品と正しく相互運用できない機能を特定します。これらの制限事項を慎重に確認してください

* Cloud Sync は中国ではサポートされていません。
* 中国以外にも、 Cloud Sync データブローカーは次の地域ではサポートされていません。
+
** Azure US 政府
** Azure US DoD
